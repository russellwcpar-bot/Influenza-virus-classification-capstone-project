# Influenza-virus-classification-capstone-project
For this project I utilized a public Kaggle dataset of electron microscopy images to build a Convolutional Neural Network (CNN) capable of distinguishing Influenza from other virus types. The process began with data preprocessing, applying Gaussian blur for noise reduction, and resizing. I initially achieved a high accuracy score, which evaluation metrics revealed was a symptom of severe class imbalance and majority class bias. To correct this I refined the training process by introducing Dropout layers, Early Stopping, and Data Augmentation, alongside strategies to rebalance how the model weighted the minority class. Ultimately, I learned that building a  machine learning model requires looking beyond raw accuracy to test for fairness, bias, and generalization.
